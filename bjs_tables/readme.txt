Mortality in Local Jails, 2000–2019 – Statistical Tables   NCJ 301368	
	
This zip archive contains tables in individual  .csv spreadsheets	
from Mortality in Local Jails, 2000–2019 – Statistical Tables   NCJ 301368.  The full report including text and graphics in pdf format is available from: https://bjs.ojp.gov/library/publications/mortality-local-jails-2000-2019-statistical-tables	
	
This report is one in a series.  More recent editions may be available.  To view a list of all in the series go to:	
https://bjs.ojp.gov/library/publications/list?series_filter=Mortality%20in%20Local%20Jails%20and%20State%20Prisons
